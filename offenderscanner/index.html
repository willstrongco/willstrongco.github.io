What's Offender Scanner?

OffenderScanner App provides users with information about nearby sex offenders registered by each state's Sex Offender Records.

The purpose of this app is to provide users with the ability to search for registered sex offenders. If you would like further information, have any education or questions then please visit Sex Offender Records site in each state.

Features

  • Find nearby sex offenders registered by each state's sex offender registry
  • Search a town or city to see if the area is safe before you go there
  • Features essential and simple functions, but gets the job done with ease of use.
  • Be aware of your neighborhood for your own safety
